### 项目介绍

咳咳，这个项目是以前上课水的作业和论文，当然是不维护的啦。
开源出来给大家参考，希望能帮到大家。

总之，这个小项目实现了一个在scikit-learn库中内置的load_digits数据集上的手写数字识别系统。
分别使用了支持向量机分类器SVC、全连接神经网络DNN、卷积神经网络CNN进行了模型训练和预测,均取得了不错的效果。

### 技术栈：

- Python
- tensorflow
- keras
- scikit-learn
- numpy
- matplotlib

### 运行环境：

额呵呵，这个是很久以前的项目了，不知道运行环境是什么样的。
